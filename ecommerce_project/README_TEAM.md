1. Open browser
2. Visit: `http://192.168.0.101:5000`
3. Start using! 🎉

### **Option 2: Developers (API Integration)**
```python
import requests

# Get dashboard stats
response = requests.get('http://192.168.0.101:5000/api/stats')
data = response.json()
print(data)

# Predict churn
prediction = requests.post(
    'http://192.168.0.101:5000/api/predict-churn',
    json={'total_spend': 500, 'order_count': 5, 'days_since_order': 30}
)
print(prediction.json())
```
